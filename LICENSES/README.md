# License texts

This directory stores reusable license texts used by package-specific releases.

Repository/package layout after the MIT-with-Unity-exception change:

- `../LICENSE.md` - repository default MIT License.
- `MIT.txt` - canonical copy used by MIT npm packages during `prepack`.
- `Stinger-Royalty-Free-EULA-1.0.md` - English convenience text copied from the previous root `LICENSE.md`.
- `Stinger-Royalty-Free-EULA-1.0.ru.md` - Russian authoritative text copied from the previous root `LICENSE.ru.md`.

Do not replace the Stinger files with a rewritten summary. Preserve the exact
existing EULA text unless the license itself is intentionally revised.
