# @romanilyin/canonicalpath

Public TypeScript package for deterministic lexical CanonicalPath operations, best-effort CanonicalFS helpers, and Unity MCP path adapter utilities.

- Root export: `@romanilyin/canonicalpath` for lexical CanonicalPath.
- Subpath exports: `@romanilyin/canonicalpath/canonicalpath`, `@romanilyin/canonicalpath/canonicalfs`, and `@romanilyin/canonicalpath/unity-gateway`.
- License: `LicenseRef-Stinger-Royalty-Free-EULA-1.0`.
- Canonical repository: `https://github.com/romanilyin/canonicalpath`.

`CanonicalPath` is lexical-only. Use the Go `canonicalfs` daemon for security-sensitive filesystem I/O.
